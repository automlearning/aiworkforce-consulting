4. Adding a New Static Project (Next.js)

For projects built with Next.js that will be hosted under /showcase/PROJECT-NAME/

1

Configure the project for static export

In your project's next.config.ts:

import type { NextConfig } from "next";

const nextConfig: NextConfig = {
  // Set the base path for the showcase subfolder
  basePath: '/showcase/YOUR-PROJECT-NAME',

  // Enable static export
  output: 'export',

  // Trailing slashes for clean URLs
  trailingSlash: true,

  // Unoptimized images for static export
  images: {
    unoptimized: true,
  },
};

export default nextConfig;
2

Build the project

cd "C:\Users\life6\OneDrive\50. Claude Code\XX. Your Project\project-folder"
npm run build

This creates an out/ folder with the static files.

3

Copy build output to showcase folder

# Copy the 'out' folder contents to the showcase
xcopy /E /I /Y "out" "C:\Users\life6\OneDrive\50. Claude Code\01. AiworkforceLandingPage\showcase\YOUR-PROJECT-NAME"
4

Add project card to showcase/index.html

Edit 01. AiworkforceLandingPage\showcase\index.html and add a new card in the projects-grid:

<!-- Your New Project -->
<div class="project-card">
    <div class="project-image">
        <img src="https://images.unsplash.com/photo-XXXXX?w=600&h=400&fit=crop"
             alt="Project Name">
    </div>
    <div class="project-content">
        <span class="project-category">Category</span>
        <h3>Project Name</h3>
        <p>Brief description of the project and its features.</p>
        <div class="project-tags">
            <span>Tag1</span>
            <span>Tag2</span>
            <span>Tag3</span>
        </div>
        <a href="your-project-name/" class="btn btn-secondary">View Project</a>
    </div>
</div>
5

Commit and deploy

cd "C:\Users\life6\OneDrive\50. Claude Code\01. AiworkforceLandingPage"

# Stage changes
git add showcase/

# Commit
git commit -m "Add YOUR-PROJECT-NAME to showcase"

# Push to GitHub
git push

# Deploy to Netlify
netlify deploy --prod
Done! Your project is now live at www.aiworkforce.consulting/showcase/your-project-name/

5. Adding an External Project (Separate Subdomain)

For projects hosted on their own subdomain (like Taurai at taurai.aiworkforce.consulting)

1

Deploy the project to its own Netlify site

Create a new Netlify site for the project with its own subdomain.

2

Add project card with external link

In showcase/index.html, use a full URL and target="_blank":

<a href="https://your-project.aiworkforce.consulting"
   class="btn btn-secondary"
   target="_blank">View Project</a>

7. Troubleshooting

Push rejected - remote has changes

git stash
git pull --rebase
git push
git stash pop

Images not loading in static export

Make sure images.unoptimized: true is set in next.config.ts

Links not working after deploy

Check that basePath matches the folder name in showcase/

404 on page refresh

Ensure trailingSlash: true is set for proper static file routing
